# Running Unit Tests

First setup ubuntu.
Install packages with:

```
    sudo apt-get install libnss3\
        libnspr4\
        libatk1.0-0\
        libatk-bridge2.0-0\
        libcups2\
        libxkbcommon0\
        libgbm1\
        libpango-1.0-0\
        libcairo2\
        libatspi2.0-0
```

Next cd to `es-tests` and type : `npm install`
Finally to run test type : `npm test`
